SELETORES:

Definem em qual/is elemento/s queremos aplicar uma propriedade CSS.


Seletor de TAG/Tipo:

Busca elementos por uma tag HTML específica.

Exemplo:

Olá!

Seletor de ID:

Busca elementos através do atributo ID. Temos de usar IDs únicos em cada página HTML. Usa (#)

Exemplo:

Seletor de id


Seletor de Classe:

Busca elementos através do atributo CLASS. Uma mesma classe pode ser usada em vários elementos diferentes, ao contrário do elemento ID, que deve ser único para cada elemento. É possível também colocar mais de uma classe no mesmo elemento. Usa (.)

Exemplo:

Seletor universal:

Seleciona todos os elementos HTML. Usa (*)

Exemplo:


Seletor de atributo:

Seleciona os elementos que possuem um atributo específico em nosso documento HTML, também sendo possível buscar atributos com um valor específico. É possível selecionar mais de um atributo por elemento. Usa ([atributo]).

Exemplo:

  • Netflix
  • Prime
  • Google
  • Disney+
